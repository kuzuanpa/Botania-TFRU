# Botania-TFRU

Fork of Botania adapted for the TFRU environment.

## Fork focus
This fork concentrates on TerraFirmaCraft/TFRU compatibility, gameplay rebalance, crash prevention, and cleanup of upstream integration points that are not useful for this pack.

## Notable changes
- Added and refined TFRU environment detection plus compatibility work for TFC item unification and GT/TFC heating behavior.
- Rebalanced mana generation, armor costs, weapon damage, and several item or flower behaviors to better fit TFRU progression.
- Fixed many gameplay bugs and crashes, including render issues, ore finder lens problems, flight bar display problems, and null-packet / NPE crash cases.
- Adjusted or restored feature behavior for items and mechanics such as Flight Tiara crafting, TerraPick interactions, Rainbow Rod, Odin Ring, Rafflowsia+, and Dandelifeon behavior.
- Removed official update checks and simplified version / jar naming for local packaging.
- Added small optimization and cleanup passes alongside recipe and value changes.
